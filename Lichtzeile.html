Lichtzeile

 
 

Lichtzeile image 0

The Lichtzeile is a 9m long LED sign that is displaying literature in public space since 1996. Since the Lichtzeile is also a message board (and was so since its inception, when the internet used cables), I had the opportunity to write some code for it and also a poem-like text that blinked through the night.
